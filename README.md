# Калькулятор расчета рисков солеотложения по Оддо-Томсону


   [![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
   [![Flet](https://img.shields.io/badge/Flet-GUI-green.svg)](https://flet.dev/)
   [![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)


Приложение для расчета рисков отложения неорганических солей в нефтяных скважинах на основе методики Оддо-Томсона с применением численных методов исследования.

## 📋 Описание

Приложение позволяет рассчитать максимально возможную массу осадков различных солей (кальцит, барит, целестин, ангидрит, бассанит, гипс) при заданных термобарических условиях и ионном составе пластовой воды.

Расчет производится с учетом:
- Термобарических условий (температура, давление)
- Ионного состава воды
- Параметров работы скважины (дебит, обводненность, газовый фактор)
- Информации о содержании CO₂ в газе и нефти

## 🔬 Физическая основа

Расчет выполняется по методике Оддо-Томсона с использованием следующих основных этапов:

1. **Пересчет единиц измерения**
   - Температура: °C → °F (Фаренгейт)
   - Давление: атм → psi
   - Концентрации ионов: мг/л → моль/л

2. **Расчет ионной силы раствора**
   ```
   I = 0.5 × Σ(cᵢ × zᵢ²)
   ```
   где cᵢ — молярная концентрация иона, zᵢ — заряд иона

3. **Расчет индекса насыщения (SI)**
   - Для кальцита: учитывается наличие/отсутствие газовой фазы
   - Для сульфатных солей: расчет с учетом константы стабильности сульфатных комплексов

4. **Итеративный расчет масс осадков**
   - Метод деления пополам для определения максимального количества каждой соли
   - Последовательное осаждение с учетом конкурирующих реакций (1000 шагов)

### Границы применимости методики

| Параметр | Диапазон |
|----------|----------|

### Результаты расчета

После нажатия кнопки **"Рассчитать"** приложение выводит:

| Показатель | Единицы | Описание |
|------------|---------|----------|
| Масса кальцита | г/л | Максимально возможный осадок CaCO₃ |
| Масса барита | г/л | Максимально возможный осадок BaSO₄ |
| Масса целестина | г/л | Максимально возможный осадок SrSO₄ |
| Масса ангидрита | г/л | Максимально возможный осадок CaSO₄ |
| Масса бассанита | г/л | Максимально возможный осадок CaSO₄·0.5H₂O |
| Масса гипса | г/л | Максимально возможный осадок CaSO₄·2H₂O |
| **Общая масса солей** | **г/л** | **Суммарная масса всех осадков** |

Интерпретация индекса насыщения (SI):
- **SI > 0**: Раствор пересыщен, возможно выпадение осадка
- **SI = 0**: Равновесное состояние
- **SI < 0**: Раствор недонасыщен, осадок не выпадает

## 🔧 Технические детали

### Основные функции расчета

- `calc_I(c)` — расчет ионной силы раствора
- `calc_SI_calcite(...)` — расчет индекса насыщения кальцита
- `calc_SI_sulfate(salt, c, T, P)` — расчет индекса насыщения сульфатных солей
- `get_max_dc(...)` — определение максимального количества осадка методом деления пополам
- `calculate_masses(...)` — главный расчет с учетом конкурирующих реакций

### Алгоритм последовательного осаждения

1. Расчет начальных индексов насыщения для всех солей
2. Определение списка пересыщенных солей (SI > 0)
3. Разбиение максимального осаждения на 1000 шагов
4. На каждом шаге:
   - Проверка текущего SI для каждой соли
   - Вычитание ионов из раствора при SI > 0
   - Накопление массы осадка
5. Перевод молярных количеств в г/л
